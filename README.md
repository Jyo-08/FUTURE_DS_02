# Customer Retention Intelligence

An interactive Business Intelligence dashboard for customer churn analysis built using Python, React, and Tailwind CSS.

---

## Live Demo

https://customer-retention-intelligence-7u4y.onrender.com/

---

## Overview

Customer Retention Intelligence is a Business Intelligence dashboard designed to help organizations analyze customer churn, understand business performance, and generate actionable insights from customer data.

The project combines a Python analytics engine with a React frontend to transform raw customer records into meaningful KPIs, behavioral analytics, revenue insights, and executive recommendations.

---

## Features

### Executive Dashboard

* Hero Dashboard
* KPI Cards

## Installation

Clone the repository:

```bash
git clone https://github.com/Jyo-08/FUTURE_DS_02.git
```

Navigate to the frontend:

```bash
cd frontend
npm install
npm run dev
```

Run the Python analytics pipeline:

```bash
cd python
python analysis.py
```
